# Module Generator

Yeoman generator for ES6 NPM modules and beyond. Uses Mocha for tests and Babel for ES6+ support.

### Install

```shell
npm install -g yo
npm install -g generator-es6-mocha-npm-module
```

### Usage

```shell
mkdir somedir && cd $_
yo es6-mocha-npm-module
```

The rest is magic.
